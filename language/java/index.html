개요

자바에 대해 알기 위해 오셨나보군요. 환영합니다! 자바는 1995년에 개발된 프로그래밍 언어로, 특히 컴파일 언어로 분류됩니다.

자바는 매우 많이 쓰이는 프로그래밍 언어 중 하나이죠. 자바의 장점 중 하나가 바로 Windows, Linux 등의 플랫폼에 독립적이라는 것이거든요! 자바 코드는 자바 가상 머신(JVM; Java Virtual Machine)에 종속적이기 때문에 JVM을 설치할 수 있는 플랫폼이라면 정상적으로 실행 가능하답니다.

그럼 자바는 무엇을 할 때 쓰일까요? 가장 대표적으로는 모바일 앱 개발, 웹 어플리케이션 / 웹 서버 개발이 있지만 데스크탑 어플리케이션 개발, 빅데이터 등등 많은 곳에서 사용되고 있답니다.

class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
