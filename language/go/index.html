Go?

이름이 Go라니, 특이하죠? 이 언어는 2012년에 발표된 따끈따끈한 프로그래밍 언어예요. 그 중에서도 컴파일 언어이죠.

이름이 '가다'여서 검색하기 힘들 거예요. 그러니 검색할 땐 'go lang'으로 하는 걸 추천드려요!

Go는 단순하면서도 실용적인 언어예요. 키워드(예약어)가 25개 밖에 되지 않는다는 것에서 그러한 점을 알 수 있죠. Go는 분산 네트워크 서비스, 클라우드 네이티브 개발 그리고 기존의 인프라 대체 등에 특히 더 좋아요.

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
